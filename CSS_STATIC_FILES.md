## **Current Settings (in settings.py)**

```python
STATIC_URL = "static/"
STATIC_ROOT = BASE_DIR / "staticfiles"  # Production
STATICFILES_DIRS = []  # Add custom dirs if needed
```

**What this means:**
- `STATIC_URL` = URL prefix for static files (`/static/`)
- `STATIC_ROOT` = Where Django collects static files for production
- `STATICFILES_DIRS` = Additional directories to collect from

## **For Production (PythonAnywhere)**

### **Step 1: Collect Static Files**
```bash
cd ~/tenancy-management-system/tenancy_system
python manage.py collectstatic --noinput
```

This command:
- Gathers all Django admin CSS/JS
- Collects any custom CSS/JS from your app
- Places them in `staticfiles/` directory

### **Step 2: Configure Web Server**
In PythonAnywhere Web tab:
- URL: `/static/`
- Directory: `/home/yourusername/tenancy-management-system/tenancy_system/staticfiles`

This tells the web server to serve files from `staticfiles/` directory directly.
